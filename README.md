# أنهال محمد - البروفايل الشخصي

- **نظام أورا (Ora)**: منصة ذكاء اصطناعي للتشخيص الإشعاعي
- **منصة لبنى (Labani)**: حل رقمي وقائي للوعي الصحي

## التقنيات المستخدمة

- HTML5
- CSS3
- Google Fonts (Tajawal, Cairo)
- Responsive Design
